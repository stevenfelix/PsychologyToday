Beginning with our early relationships with our family, followed by teachers and friends, we develop our own unique attachment style. Our practice is focused on empowering clients to improve the meaningful connections in their lives, with themselves as well as their significant others, through psychotherapy. It is as a powerful tool for self-transformation, come about through expanding self-understanding and reducing difficult symptoms.

                    <div class="statementPara">

I support clients in the process of discovering new ways to relate to their challenges and themselves so they feel better able to manage challenges and reconnect with others.
        </div>

        function contactBar() {
            var windowWidth = $(window).width();
            if (windowWidth > 768) {
               var height = $('#profHdr').height() + 115;
                if (window.pageYOffset > height) {
                        showContactBar();
                    } else if (window.pageYOffset <= height) {
                        hideContactBar();
                    }
            } else {
                var height = $('#profHdr').height() + 40;
                if (window.pageYOffset > height) {
                    showContactBar();
                } else if (window.pageYOffset <= height) {
                    hideContactBar();
                }
            }
        }
        function setContactBarPosition() {
            var offset = $('#contactBar').outerHeight() - $('#headerShadow').outerHeight();
            $('#contactBar').css({
                "top": '-' + offset + 'px'
            });
        }

            var offset = $('#contactBar').outerHeight();
            $('#headerShadow').css({
                "top": offset + 'px'
            });
        }

            $('#headerShadow').css({
                "top": '0px'
            });
